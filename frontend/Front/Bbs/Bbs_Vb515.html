사용가이드

[신청] [신청] 배송대행 신청방법




   배송대행 신청 전 확인!


- 제품구입: 원하는 상품이 있는 해외 쇼핑몰에 회원 가입하여 상품을 구매
- 센터주소 입력: 구매시 받는 주소(Shipping Address)란에 직플 물류센터 주소를 입력뒤 구매 
  [센터주소입력방법]
  센터주소: [메인] > [센터주소]  혹은 [마이페이지] > [나의 배송대행지]

  *주의사항: 상품 주문 시에 배송가능 여부, 수입금지품목 및 관부가세 주의*


    신청서 입력 (소량추천)



Step 1. 배송대행 신청 페이지 접속




페이지 :  [대행신청] >  [배송대행 신청]






Step 2. 물류센터/운송방식 선택








Step 3. 받는 사람 정보 입력




- 받는사람 :  개인/사업자/외국인 선택후 성함/상호, 연락처 입력
- 통관부호입력/체크 : 통관부호 입력뒤 유효성 체크  [발급안내 보러가기] 
- 주소 : 우편번호 검색 추가
- 배송 요청사항 : 택배사측 전달사항 선택/직첩입력





Step 4. 상품 정보 입력


- 쇼핑몰 사이트 : 해외구매 쇼핑몰 주소지 입력 (예시: taobao.com, 1688.com)
- 브랜드 : 제품 브랜드 입력
- 주문번호 : 해외쇼핑몰 결제뒤 생성된 주문번호 
- 트래킹번호 : 구매쇼핑몰 송장번호 (필수입력)
- 통관품목/상품명영문 : 통관품목 검색 혹은 직접선택 뒤 입력, 상품명 자동번역 등록 (필수입력) 
- 수량 : 총구매수량  (필수입력)
- 합계 : 총구입금액. 총금액 150불 초과시 자동 간이신고 체크 (필수입력)
- 색상/사이즈 : 구매 옵션 입력  (필수입력)
- 상품 URL : 구매쇼핑몰 url 입력  (필수입력)
- 이미지 URL : 구매제품 쇼핑몰 이미지 URL (검수시 필수입력)
- 판매처명/판매처주문번호/팬매금액 단가: 판매자회원 편의 기능 


 상품정보 쉽게 불러 입력해 보세요.
 
1.등록상품 불러오기 : 자주 구매하는 제품, 나의 등록상품에 등록뒤, 바로 선택 등록    
2.상품URL 가져오기 : 쇼핑몰 URL 입력 뒤, 상품 옵션선택 등록 
3.오더내역 복사 : 쇼핑몰 구매내역 있는경우, 드래그 복사하여 간편등록 (중국어로 등록)
4.재고 불러오기 : 센터재고제품 출고요청 




Step 5. 결제방식 선택


- 자동결제 : 선택시 입고완료 즉시 예치금 자동결제 출고대기 
- 수동결제: 결제대기에서 고객님 직접 수동결제






Step 6. 부가서비스 선택


- 필요한 서비스 체크 선택  




Step 7. 신청서 접수



접수확인 :  [마이페이지] > 배송대행 > 임시저장 / 접수신청
